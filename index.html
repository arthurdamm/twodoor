Failure Based Repetition

Our app is designed to help you learn/remember any topic that is important to you. Whether it is remembering coworker's names, key coding concepts, or any information under the sun, we want to help you (and ourselves) transfer concepts into long-term memory. Through failure-based repetition, you can efficently encode any info in your brain the most impactful way possible.

Quick, fun to use

When designing this app, we knew it was important to create a fun gameplay experience. Through fun transitions, animations, and good feedback, we created a polished experience that all will want to use for brain-training needs.

Features

Adaptive Algorithm

Using a variation of the Leitner system, we created a dynamic card-selection algorithm that shows a subset of the entire deck, giving the user a good chance to succeed through repetition.

Pre-Built Decks

So far, the decks are pre-built to show the capabilities of the game. Some are only used to show how the algorithm can teach you completely random information within seconds. User-made decks on the way!

Intuitive UX

Keeping the user in mind, a polished display was crucial for creating a fun experience for all. Custom animations make for a unique experience that provides seemless positive and negative reinforcement.
